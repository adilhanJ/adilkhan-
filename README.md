 while True:
Это значит: “повторяй всё ниже бесконечно”.
Программа будет крутиться, пока её не остановят.

 — меню
Программа каждый раз показывает тебе варианты:

1 — сказать «крокодилобомбордиро»
2 — показать число 67
0 — выйти

 choice = input(...)
Программа ждёт, что ты введёшь с клавиатуры.
Например: 1, 2 или 0.
Всё, что ты ввёл, сохраняется в переменную choice.

 Проверка выбора (if / elif / else)

Дальше программа смотрит, что ты ввёл:

Если ввёл "1"
 напечатает: крокодилобомбордиро!
Если ввёл "2"
напечатает: Число: 67
Если ввёл "0"
напечатает сообщение и остановит цикл (break)
→ программа закончится
Если ввёл что-то другое
 напишет: “Неверный выбор”

 Что происходит дальше?
Если ты не выбрал 0, программа возвращается в начало и снова показывает меню.

 Коротко:

цикл работает постоянно
ты выбираешь действие
программа делает это действие
и снова спрашивает, пока не нажмёшь 0
